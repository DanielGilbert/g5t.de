Tools

SmallTune

SmallTune is a tiny audio player for Windows. I wrote this back in 2009. It uses SQLite and allows you to play local files, as much as listening to internet radio stations.

dgMicMute

dgMicMute is a small program that allows you to mute your microphone with a shortcut in Windows. This feature is now integrated into Windows. As such, this tool is obsolete.

kenBuddy

kenBuddy is a tiny helper for kenjo.io - if you are not legally required to track correct times, this extension will help you make your life easier by filling the attendance sheet out for you. You can get it for Google Chrome and Firefox.
